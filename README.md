# 광주 서구 마을안전지도 웹앱

광주 서구 18개 동 안전지도를 바탕으로 시민참여단이 현장 의견을 등록하고, 관리자가 실시간으로 종합·관리하는 웹앱입니다.

## 웹앱 구성

- 시민참여단 앱: 저장소 최상위 `index.html`
- 관리자 앱: `admin/index.html`
- 공동 자료 저장소: Google Cloud Firestore
- 관리자 로그인: Firebase Authentication의 Google 로그인

## 시민참여단 기능

- 18개 행정동 선택
- 여성안심귀갓길·여성안전골목·택배보관함·비상생리대 자판기 확인
- 일반 배경지도와 동별 대표 학교·공원·공공기관·역 표시
- 지도에서 취약 위치 선택
- 취약 유형과 간단한 의견 등록
- 사진 촬영 및 개인정보 입력 없음

## 관리자 기능

- 관리자 Google 계정 로그인
- 18개 동 전체 또는 동별 의견 실시간 확인
- 접수·검토 중·지도 반영·보류 상태 관리
- 모바일과 PC에서 안심귀갓길·안전골목 선과 택배보관함·생리대 자판기 위치 추가·수정·삭제
- 시민참여단 의견의 내용·유형·행정동·위치 수정 및 삭제·복구
- 안전시설 변경사항을 시민참여단 지도에 실시간 반영
- CSV 내려받기
- 관리자 지도 인쇄

## Firebase 연결

자세한 설정은 `FIREBASE_설정안내.md`를 확인합니다. Firebase 연결 전에는 같은 브라우저에서 시범 기능을 확인할 수 있습니다.
